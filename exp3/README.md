MNIST Digit Classification (Neural Network)

This project uses a Neural Network (ANN) built with TensorFlow & Keras to classify handwritten digits from the MNIST dataset (0–9).

Key Points

Dataset: MNIST (28×28 grayscale images)

Model: Dense Neural Network (784 → 128 → 10)

Activation: ReLU, Softmax

Optimizer: Adam

Loss: Sparse Categorical Crossentropy
